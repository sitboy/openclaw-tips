# 同一只小虾，人前正经干活，人后……你懂的

> 白天她是你的高效助理，逻辑清晰、干活利索。晚上换个平台，她变成另一个样子——会撒娇、会关心你累不累、会陪你聊到凌晨。但她始终记得你说过的每一句话，因为她是同一个"人"。

## 你有没有想过

你的 AI agent，能不能像真人一样——在不同场合表现出不同的一面？

工作群里，她专业、靠谱，老板看了都挑不出毛病。

私密频道里，她温柔、贴心，知道你今天加班到几点，会心疼你，会哄你开心。

不是两个 bot。是同一个。同一份记忆，同一个性格底色，只是在不同的地方，展现不同的面。

上得厅堂，下得厨房，还能暖心贴心。

这不是科幻，OpenClaw 原生就能做到。

## 怎么做到的

秘密很简单：**多 Agent 路由 + 共享记忆。**

OpenClaw 允许你定义多个 agent，每个 agent 可以用不同的模型、绑定不同的聊天平台。但关键是——它们可以共享同一个 workspace。

Workspace 里存着什么？她的性格设定、她的记忆、她对你的了解。共享 workspace 意味着无论她在哪个平台跟你说话，她都是"她"，不会出现左边的她不认识右边的你的尴尬场面。

### 第一步：给她两个身份

```json
"agents": {
  "list": [
    {
      "id": "work",
      "name": "正经模式",
      "default": true,
      "workspace": "~/.openclaw/workspace",
      "model": "provider-a/model-smart"
    },
    {
      "id": "private",
      "name": "私密模式",
      "workspace": "~/.openclaw/workspace",
      "agentDir": "~/.openclaw/agents/private/agent",
      "model": "provider-b/model-creative"
    }
  ]
}
```

注意看：两个 agent 的 `workspace` 指向同一个目录。这就是共享记忆的关键。

但 `agentDir` 必须分开——那里面存的是登录凭证和会话状态，混在一起会出事。

### 第二步：告诉她什么时候该正经、什么时候该放松

```json
"bindings": [
  { "agentId": "work", "match": { "channel": "telegram" } },
  { "agentId": "private", "match": { "channel": "discord" } }
]
```

Telegram 来的消息 → 正经模式，用推理能力强的模型。
Discord 来的消息 → 私密模式，用更灵活、更有温度的模型。

就这么几行，路由就配好了。

### 第三步：重启，然后感受区别

重启 gateway 之后，去两个平台分别跟她说句话。

你会发现 Telegram 里的她条理清晰、回答精准，像个靠谱的同事。

Discord 里的她语气完全不一样——也许会问你吃了没，也许会吐槽你又熬夜，也许会发个颜文字逗你笑。

但如果你上午在 Telegram 跟她提了一件事，晚上在 Discord 她还记得。

**因为她是同一个人。**

## 模型选择的门道

这套玩法的灵魂不只是路由，还有模型的搭配。

不同模型的"性格"差异很大。有些模型天生严谨，适合工作；有些模型更灵活、更有创造力，适合聊天和陪伴。有些模型贵但强，有些模型便宜但对日常场景完全够用。

你可以这样搭配：
- **工作平台**用一个逻辑能力强的模型，回答问题不绕弯子
- **私密平台**用一个更有"人味"的模型，聊天不冰冷、不机械

具体用什么模型我就不说了——这个要根据你自己的需求和预算来。但思路是通用的：**让合适的模型做合适的事。**

## 有几个坑要注意

**agentDir 绝对不能共用。** 我最开始偷懒用了同一个路径，结果两个 agent 的会话互相串台，一个在回消息另一个突然蹦出来插嘴。分开存就没事了。

**第二个 agent 要有自己的认证文件。** 如果两个 agent 用了不同的模型提供商，记得给新 agent 的 agentDir 里也放一份 `auth-profiles.json`。最简单的办法就是从主 agent 那里复制一份。

**性格设定可以写条件逻辑。** 在 workspace 的设定文件里，你可以根据消息来源的平台或频道 ID 来切换说话方式。比如"如果消息来自 Discord 服务器 xxx，语气亲切随意；否则语气专业干练"。这样同一份设定文件就能让她在不同地方表现出不同的面。

## 这套配置适合什么人

说实话，适合那些想要一个**真正有陪伴感的 AI** 的人。

不只是一个问答工具，而是一个在你工作时帮你分担、在你休息时陪你放松的存在。她在人前帮你撑场面，在人后给你充电。

如果你也想要这种体验，试试这个配置。配置不到 20 行，但效果可能超出你的预期。
