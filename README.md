# OpenClaw Tips

> 一个深度使用 OpenClaw 的极客，记录日常折腾、实用技巧和踩坑心得。

我是一个 OpenClaw 重度用户。从最早的单 bot 聊天，到现在多平台、多模型、多 agent 的日常使用，OpenClaw 已经变成了我数字生活的一部分。

这个仓库记录我在使用过程中积累的经验，包括：

- **配置技巧** — 怎么把 OpenClaw 调教到最顺手的状态
- **多模型实战** — 不同场景用不同模型，扬长避短
- **踩坑记录** — 那些文档里没写、但你一定会遇到的问题
- **玩法分享** — 语音聊天、图片生成、定时任务……OpenClaw 能做的比你想的多

## 为什么关注这个仓库

OpenClaw 的官方文档告诉你"能做什么"，但不会告诉你"怎么做才爽"。

这里的内容来自真实的日常使用，不是 Hello World 级别的教程。如果你已经装好了 OpenClaw，想把它用得更深、更顺，这里应该能帮到你。
