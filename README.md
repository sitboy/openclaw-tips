# 🦞 OpenClaw 野路子手册

**把 AI Agent 玩出花来。**

我每天跟自己的 AI agent 聊天、干活、协作。Discord、Telegram 多平台跑，多模型混着用，语音、图片、定时任务全都上。这个仓库记录我一路折腾下来那些文档里不会告诉你的东西。

## 这里有什么

不是入门教程，不是官方文档的复读机。

这里是一个重度用户的实战笔记——**怎么把 OpenClaw 从"能用"搞到"好用"再到"离不开"。**

- 🔀 多模型混用 — 不同场景自动切换最合适的模型
- 🎙️ 语音对话 — 用免费/低价 API 实现语音收发
- 🧠 记忆系统 — 让你的 agent 真正"记住"你
- ⚡ 性能调优 — 省钱、提速、减少翻车
- 💀 踩坑实录 — 我翻过的车，你不用再翻
